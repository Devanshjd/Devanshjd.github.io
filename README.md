# Devansh Jamdar Portfolio

Standalone animated portfolio website built from Devansh Jamdar's CV and the local Jarvis project.

Open `index.html` directly in a browser, or serve this folder with any static file server.

## Contents

- `index.html` - portfolio content and layout
- `styles.css` - responsive visual system and animation
- `script.js` - scroll reveal, project filters, Jarvis screenshot switching
- `assets/` - local Jarvis screenshots and the generated UI concept reference
