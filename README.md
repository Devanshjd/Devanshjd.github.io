# Devansh Jamdar Portfolio

Personal cybersecurity portfolio for **Devansh Jamdar**, built as a sleek animated static website for sharing on LinkedIn, WhatsApp, and job applications.

The site positions Devansh as a **Cyber Security Graduate / Security Engineer** with practical experience in vulnerability assessment, network security, security-focused machine learning, IoT systems, and AI automation.

## Live Preview

Open locally:

```text
index.html
```

Or serve the folder with any static server.

## Highlights

- Strong cybersecurity-focused hero section
- Animated dark UI with responsive design
- Project case-study cards with detailed popups
- Featured Jarvis AI Assistant project
- Security Projects in Action section
- Skills section focused on recruiter keywords
- Education and experience timeline
- Contact dropdown with email, phone, and GitHub
- Downloadable CV asset

## Featured Projects

- **Jarvis AI Assistant** - Local AI assistant runtime with Electron, React, Python, plugins, task memory, and automation controls.
- **Phishing Website Detection** - Machine learning workflow for identifying suspicious URLs using security-focused features.
- **AI Safety Gridworlds** - Reinforcement learning environments for analysing unsafe reward-seeking behaviour.
- **ESP32 Smart Environment Monitoring** - Embedded IoT monitoring system with sensor alerts and manual override.
- **AI Dog Breed Classification** - Deep learning image classification project using TensorFlow/Keras.

## Deployment

This portfolio can be deployed for free using GitHub Pages.

Recommended repository name:

```text
portfolio
```

Expected public URL:

```text
https://Devanshjd.github.io/portfolio/
```

GitHub Pages settings:

- Source: deploy from branch
- Branch: `main`
- Folder: `/root`
